# Projet-1_JAVA
Le jeu Puissance 4 est un jeu de stratégie où deux joueurs s'affrontent pour aligner quatre de leurs pions dans une grille. Le joueur qui réussit à  aligner quatre pions en ligne gagne  la partie.  Le projet consiste à créer une version de ce jeu avec une interface graphique simple en  utilisant Java et des outils comme Swing ou JavaFX.
